# ProgressBar

## Usage

```jsx
import { ProgressBar, theme } from '@aragon/ui'

const App = () => <ProgressBar color={theme.accent} progress={0.3} />
```

## Properties

### `color`

- Type: `String`
- Default: `accent` color from theme

### `progress`

- Type: `Number`
- Values: from `0` to `1`
